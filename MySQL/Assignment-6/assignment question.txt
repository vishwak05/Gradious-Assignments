You are provided with a dataset named transactions.csv, containing information about customer transactions.
Your task is to perform various analyses on this dataset using the Pandas SQL interface.

You are required to write Python code to achieve the following tasks:

    3. Return results sorted by total spending in descending order.
    4. Calculate the average spending per customer for each month in the year 2023.
    5. Identify the customer with the highest total spending and the customer with the lowest total spending.

Dataset Description: Link

File Name: transactions.csv

Columns:
